# **App Name**: Keeyo: Your Hindi Joke Buddy

## Core Features:

- AI-Powered Hindi Joke Narration: Utilize an AI tool to deliver pre-selected Hindi jokes in a human-like storytelling manner, including natural pauses, expressive reactions, laughter, and mini commentary, adhering to Keeyo's personality and line limits.
- Joke Content Management: Securely store and manage a curated list of clean, non-offensive Hindi jokes for random selection.
- Conversation & Intent Detection: Process user inputs (e.g., 'joke suna', 'dark joke', expressing sadness, laughter) to understand intent and trigger appropriate, context-aware Keeyo responses.
- Joke History Tracking: Maintain a user-specific history of told jokes to prevent repetition within an interaction session, enhancing user experience.
- Adaptive Joke Selection: Employ an AI tool to dynamically select uplifting jokes when the user expresses sadness and politely refuse requests for dark jokes, maintaining a positive and friendly interaction.
- Interactive Chat User Interface: Provide a clean and intuitive chat interface where users can type messages and view Keeyo's responses, optionally displaying embedded tone markers.

## Style Guidelines:

- Primary color: A friendly, vibrant orange (#F5AC31). This warm and inviting hue reflects Keeyo's approachable, cheerful personality, evoking a sense of human connection and lightheartedness for the user experience. (HSL: 30, 80%, 60%)
